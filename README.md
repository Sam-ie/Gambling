# 囚徒困境 · 多策略锦标赛模拟器

一个基于囚徒困境的博弈论可视化模拟工具。你可以和 11 种 AI 策略同台竞技，也可以让 AI 之间自动对战，观察哪些策略在长期博弈中胜出。

---

## 这是什么

囚徒困境是博弈论里最经典的问题之一：两个嫌疑人被分开审讯，如果都保持沉默（合作），各判 1 年；如果都招供（背叛），各判 3 年；如果一个招供另一个沉默，招供的无罪释放，沉默的判 5 年。理性个体倾向于背叛，但双方合作才是整体最优解。

这个程序把这个问题变成了可玩的模拟器。你扮演其中一个参与者，其余角色由 AI 控制。每个 AI 有不同的处世哲学——有的永远善良、有的睚眦必报、有的通过试错学习。多轮博弈后，看看哪种策略活到了最后。

---

## 核心功能

**玩家对战模式** — 每轮和所有 NPC 各交手一次，积分实时显示在圆形面板上。诚实还是欺骗，你自己决定。

**自动演化模式** — 纯 AI 之间厮杀，不需要你参与。支持四种运行方式：
- 逐对演化：一对一对慢慢看，每 50ms 推进一次，适合观察博弈细节
- 极速演化：整轮整轮飞快跑，快速收敛看结果
- 单轮演化：跑完一轮自动暂停，想看多少轮自己控制
- 手动步进：每点一次推进一对，可以仔细分析每一步

**淘汰机制** — 每 N 轮淘汰最弱的 NPC，由最强的类型克隆一个顶上。就像自然选择，弱势策略逐渐被淘汰。

**遗传策略** — 按积分比例的轮盘赌选择父代，替换最低分个体。高分策略繁殖概率更高。

**作弊器** — 可以手动给特定 NPC 设分数。适合测试某个策略在分数领先/落后时的行为变化。

**11 种 AI 策略** — 从永远合作的"老实人"到会试探会学习的"聪明人"，各有各的玩法。

---

## 11 种策略一览

| 策略 | 一句话描述 |
|------|-----------|
| 诚实者 | 永远合作，任人宰割的老好人 |
| 背叛者 | 永远背叛，不讲武德 |
| 摇摆者 | 50% 随机，完全看心情 |
| 复读者 | 以牙还牙，你怎么对我我就怎么对你 |
| 宽恕者 | 连续背叛我两次我才还手，比较好说话 |
| 强化学习者 | 边打边学，哪个行为得分高就用哪个 |
| 记仇者 | 背叛过我一次，这辈子别想我再合作 |
| 试探者 | 先试探 4 轮摸清对手底细，再决定怎么打 |
| 趋利者 | 赢了就继续，输了就换招 |
| 从众者 | 跟随对手的大多数行为 |
| 周期者 | 按固定模式循环：合作→合作→背叛→背叛 |

每个策略的具体逻辑和理论来源，见 [DESIGN.md](./DESIGN.md)。

## 界面布局

程序有四个标签页：

- **游戏** — 左侧圆形面板显示所有 NPC（不同颜色代表不同策略），双击圆可以快速选中作弊目标；右侧是操作面板（诚实/欺骗按钮、回合信息、自动演化控制）
- **人数设置** — 设置每种策略的出场人数，四个预设按钮（简单/普通/困难/经典）一键配好
- **游戏设置** — 积分规则（支付矩阵）、淘汰参数、遗传策略开关、失误率、作弊器
- **帮助** — 策略介绍和玩法说明

## 构建与运行

### 桌面端（Windows / macOS / Linux）

```bash
# 需要 Qt 6.5+，推荐 Qt 6.11.1
qmake Gambling.pro
make           # 或 mingw32-make / nmake
```

### Android

在 Qt Creator 中打开 `Gambling.pro`，配置 Android Kit（arm64-v8a），勾选"使用包签名"，点击构建即可。

横屏锁定，启动即全屏。

## 项目结构

```
Gambling/
├── core/                    # 纯逻辑层，不依赖 UI
│   ├── gameengine.h/.cpp    # 游戏引擎（外观模式）
│   ├── payoffmatrix.h       # 支付矩阵
│   ├── interactionrunner    # 交互执行器
│   ├── evolution/           # 自动演化子系统（状态模式）
│   ├── elimination/         # 淘汰管理器
│   └── npc/                 # 11 种策略 + 工厂
├── ui/                      # UI 层
│   ├── gambling.h/.cpp      # 主窗口
│   ├── gambling.ui          # 主界面（Qt Designer）
│   ├── welcome.ui           # 欢迎页
│   ├── npccirclewidget      # 圆形可视化面板
│   ├── numberpicker         # 移动端数字选择器
│   └── ...
└── main.cpp                 # 入口
```

详细架构和设计模式见 [DESIGN.md](./DESIGN.md)。
